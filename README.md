# Gopika Cottage Booking

A charming mountain retreat booking application for Gopika Cottage in Uttarakhand.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Deployment

Deploy via the platform's Share → Publish option, or self-host using the build output.

## Custom Domain

To connect a domain, navigate to Project > Settings > Domains and click Connect Domain.
